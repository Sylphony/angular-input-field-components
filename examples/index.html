AngularJS Input Field Components

A simple way of creating form components with AngularJS 1.5+

To use

  1. Download the file here.
  2. Include the file in your HTML document after AngularJS has been added, e.g.
    <script src="https://cdnjs.cloudflare.com/ajax/libs/angular.js/1.6.1/angular.js></script>
    <script src="angular-input-field-components.js"></script>
  3. Include it as a dependency within angular.module, e.g.
    angular.module("app", ["inputFieldComponents"])
