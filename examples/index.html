AngularJS Input Field Components

A simple way of creating common form components with AngularJS 1.5+

Setup

  1. Download the file here.
  2. Include the file in your HTML document after AngularJS (1.5+) has been added, e.g.
    <script src="https://cdnjs.cloudflare.com/ajax/libs/angular.js/1.6.1/angular.js></script>
    <script src="angular-input-field-components.js"></script>
  3. Include it as a dependency within angular.module, e.g.
    angular.module("app", ["inputFieldComponents"])

Form fields

This module can currently create the following form field types:

  • Textbox
  • Textarea
  • Checkbox

The component that creates these fields is <field>. Within the <field> is a label element and a corresponding input element. The type attribute determines what field type to set. See the Customization section below for more details.

Customization

Attributes can be placed on the <field> component to customize it. The following is a list of these attributes.

type

Input type: String

This determines what form field element to create. Possible values are text, textarea, checkbox. If the value passed is not one of the ones specified, nothing is rendered and a warning will appear in the console.

label

Input type: String

Field types supported: Textbox, Textarea, Checkbox

This sets a label for the element.

ref-id

Input type: String

Field types supported: Textbox, Textarea, Checkbox

This assigns an id value to the input element. The value is also set on the for attribute on the label element to link them together.

classes

Input type: Object

Field types supported: Textbox, Textarea, Checkbox

This allows class to be set on the input element, label element or both. The property names of the object passed to set the label and input are label and input respectively.
